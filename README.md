# W2Day2hw
YtYt — это образовательная платформа, специализирующаяся на подготовке специалистов в сфере IT и разработке программного обеспечения. Текущая сборка включает в себя адаптивный подвал (footer) официального сайта.
​Основные разделы и навигация:
​Чему вы научитесь — информация о ключевых навыках, курсах и доступных технологиях.
​Процесс обучения — описание образовательного формата, практики и проверки домашних заданий.
​Стоимость — тарифные планы, условия подписки и оплаты обучения.
​Контакты — каналы связи со службой поддержки и менторами.
​Регистрация — точка входа для создания учетной записи студента.
